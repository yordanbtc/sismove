# SOS Sismo VE

Plataforma de reporte y localización de personas afectadas y solicitudes de ayuda ante la emergencia sísmica en Venezuela.

## Propósito del Proyecto
Este proyecto surge a partir de la necesidad urgente de organizar y canalizar la información durante la emergencia tras el terremoto reciente en Venezuela. Es una iniciativa **sin fines de lucro**, desarrollada con el objetivo de facilitar la conexión entre personas desaparecidas, encontrados y quienes requieren asistencia inmediata.

## Instrucciones de Uso

### 1. Visualización de Datos
Al ingresar a la aplicación, verás dos pestañas principales:
* **Solicitudes de Ayuda:** Aquí podrás ver los requerimientos urgentes (Agua, Refugio, Salud, etc.). Esta es la vista predeterminada.
* **Desaparecidos:** Cambia a esta pestaña para visualizar los reportes de personas cuyo paradero se desconoce o ha sido confirmado.

### 2. Cómo reportar un caso
* **Si deseas reportar a una persona:**
    1. Haz clic en el botón correspondiente para abrir el formulario de "Reportar Persona".
    2. Completa los campos solicitados (Nombre, Apellido, Ubicación, Descripción).
    3. Si posees el DNI/Cédula, márcalo en la casilla correspondiente para habilitar el campo.
    4. Envía el formulario. El sistema actualizará las estadísticas automáticamente.

* **Si necesitas solicitar ayuda:**
    1. Abre el formulario de "Solicitar Ayuda".
    2. Selecciona la categoría (Agua, Salud, Rescate, etc.).
    3. Ingresa tus datos de contacto y ubicación precisa.
    4. Envía el formulario para que sea visible en el tablero principal.

### 3. Búsqueda
En la pestaña de **Desaparecidos**, puedes utilizar la barra de búsqueda para filtrar registros escribiendo el nombre o apellido de la persona que buscas.

## Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3, JavaScript.
* **Backend/Base de Datos:** [Supabase](https://supabase.com/).
